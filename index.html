Heart Disease Classification

A machine learning model that predicts the likelihood of heart disease in a patient based on clinical features such as age, cholesterol levels, resting blood pressure, and more. Includes a full analysis of model performance, feature importance, and overfitting behavior.

Tech Stack:

  • Python - Scikit-Learn
  • Pandas - NumPy
  • Matplotlib - Seaborn

Main Objective:

  • Classify patients as at-risk or not at-risk using RandomForestClassifier
  • Analyze confusion matrix and feature importance to understand model decisions
  • Identify and address overfitting patterns

Deployment:

  • Jupyter Notebook, hosted on GitHub

Titanic Survival Prediction

A classification model predicting passenger survival on the Titanic using engineered features like extracted passenger title, family size, and cabin availability. Achieved 83% accuracy on the test set, with feature engineering meaningfully improving on raw data alone.

Tech Stack:

  • Python - Scikit-Learn
  • Pandas - Pipeline & ColumnTransformer
  • Matplotlib - Seaborn

Main Objective:

  • Engineer features (Title, FamilySize, HasCabin) to improve prediction accuracy
  • Compare Logistic Regression vs Random Forest performance
  • Evaluate with confusion matrix and precision/recall analysis

Deployment:

  • Jupyter Notebook, hosted on GitHub
